Gerador de senhas
ATENÇÂO

Se o computador estiver infectado ou um site malicioso executar scripts, eles podem acessar o localStorage do seu domínio e roubar os dados.

Não é recomendado usar este gerador para salvar suas senhas! É apenas um projeto academico!

ATENÇÂO

Funcionalidades
Gerar senha com 1 clique. Configurações: incluir letras maiúsculas/minúsculas, números, símbolos e escolher o comprimento.

Salvar senhas localmente (localStorage).

Listar senhas salvas.

Mostrar / ocultar senha.

Excluir uma senha já salva.

Copiar senha para clipboard com um click.

UI baseada em componentes Vue

Tecnologias
Vue

Vite (Para rodar no codespace)

CSS simples

localStorage para persistência local
